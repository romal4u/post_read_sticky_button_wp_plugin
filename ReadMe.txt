== Description ==

**Smart Floating / Sticky Buttons** used to get the **read status** of the post or page or any custom post if user reach at the end of post. You can get the detailed report with date and time in admin that which post or page read at which time by which user of your website. **No Coding!** 

This plugin works only for logged-in users. It will not work for the guest user.

By using this plugin users can get the status of a post or page at the front end while reading. After completion of the reading, the user can update the status of the post or page by simply clicking on the button that reads completely.

**There are four types of status are there.**

* Please read first
* If click on read first button then ask for **Set me read**
* Read completely
* Already Read

All status you can check in attached screenshot below.

Admin can view the read status report of all post or pages. There is separate menu for the same with name **Post Read Report**

== Screenshots ==

1. The icon indicates that the post or page is not read yet so read it first. 
2. The icon indicates that the user reached the end of the post or page, now the icon charged to Set me to read for update status.
3. Once click on Set me to read it will show you that status updated. 
4. If the same user goes to the same post it shows that you have already read this post.
5. Users can view the Read post report from the admin panel.

== Frequently Asked Questions ==

= Is this plugin work for guest users? =

No, right now ths plugin is only worked for the logged in user.

= Do I need coding expertise for install this plugin? =

No, you have to just install the plug in and it will work automatically without any code.

= Can user set the status of post read without scrolling the page? =

Yes, for that user has to click on read first icon and then update the status of post or page as read.

= Does it work with the page scroll? =

Yes, if user read the post or page and at the end of the post icon automatically changed for the status if Set me click.
